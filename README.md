# Chatroom 

## Add index.html 
- Add index.html with following in public directory
```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Chatroom</title>
</head>

<body>
    <div id="root"></div>
</body>

</html>
```
### Setup firebase

```bash
firebase deploy
```
